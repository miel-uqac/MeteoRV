# MeteoRV
Application de réalité virtuelle sur le thème de la prévision météorologique

## Environnement de développement
- Unity (6000.3.6f1)
- XR Interaction Toolkit (3.3.1)

## Spécificités

### Modalités de déplacements
#### Contrôleur gauche
- Déplacement continu

#### Contrôleur droit
- Rotation instantanée (_snap turn_)
- Téléportation
- Saut
